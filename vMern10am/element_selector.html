#Element or Tag selector

  • =>Element selector are patthern that tell the browser which HTML elements to apply CSS property values to.
  • =>The selected elements are known as the selector's subject.
#Syntax: 
            element{
                .......
                style-code(css-code) ;
                .......
            }

            

            LIKE ->
            p{
                ...
                ...
                ...
            }
