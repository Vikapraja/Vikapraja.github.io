#Css Display Property

  • =>The display property determines how an element appears on an web page, including ita box type (block or inline) and the layout of its child elements
  • => The display property secifies the display behavior(the type of rendering box) of an element.
  • => display property default values is 'block' or 'inline'
  • =>Syntax : display: value;
  • =>CSS display properties possible values
    1. block

      [is used to adjust one element in one row. In that case we use height,width,padding and margin . Follwing elements having display block properties by default:

      < table >, < tr >, < heading >, < p > etc.
      ]

    2. inline

      [It is used to adjust display more than one element in a row. In that case we cann't use height and width but padding and margin will be used. Following elements having display inline property by default:

      < a >, < label >, etc.
      ]

    3. inline-block

      [It is used to adjust more than one element in a row . In that case we can use height, width, padding & margin. Following elements having display inline-block property by default :

      < button >, < image >, < Input > etc.
      ]

    4. None

      [is used to remove an element from normal flow of page of page. In that case element are remove or hide from web-page. and other element will not fill hidden element's gap.]

    5. Etc.
#Syntax : 
            display: value;
