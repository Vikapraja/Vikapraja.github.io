#Group Selector:

  • =>The grouping selector is used to style all comma-separated elements with the same style.
  • =>It will be better to group the selector, to minimize the code.
  • =>To group selectors, separate each selector with a comma.
#Syntax: 
            element1,selector,element2{
                ......
                .....
                code of Css
                .........
            }
