#Border In CSS

  • =>Css Border properties allow you to specify the style,width and color of an element's border.
  • =>CSS border properties are internal of elements.
  • =>Border properties values:
    1. border-width: (top)px (right)px (bottom)px (left)px ; OR border-width: (int)px; [difault all sides]
    2. border-top: (int)px;
    3. border-right: (int)px;
    4. border-bottom: (int)px;
    5. border-left: (int)px;
    6. #NOTE:We can give a border's all values like border: (width) (style) (color) (side)
  • #Css border-styles: (values) ;
    1. dotted
    2. dashed
    3. solid
    4. double
    5. groove
    6. ridge
    7. inset
    8. outside
    9. none
    10. hidden
  • border-radious: (int)%; for rounded corner border.

  • #EXAMPLEs :

    I have borders on all sides.

    I have a red bottom border.

    I have rounded borders

    I have blue left border.
