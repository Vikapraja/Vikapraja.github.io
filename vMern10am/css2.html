#CSS Selector

=> It is used to "find" (or select) the html element you want to style.

  1. Universal Selector '*'
  2. class Selector '.'(dot)
  3. id selector '#'
  4. tag/element Selector
  5. Child selector '>'
  6. group Selector ','
#Syntax: 
            
    
                
  1. 
                    Element or tag selector :
                    
                        element{
                            .......
                            style-code(css-code) ;
                            .......
                        }

                
    2. 
                
  2. 
                class selector :
                
                    .class_name{
                        ...
                        style-code(css-code) ;
                        ...
                    }
                
    3. 
                
  3. 
                ID selector :
                    
                    #id_name{
                        ...
                        style-code(css-code) ;
                        ...
                    }
                
    4.
