#CSS Selector

=> It is used to "find" (or select) the html element you want to style.

  1. Universal Selector '*'
  2. class Selector '.'(dot)
  3. id selector '#'
  4. tag/element Selector
  5. Child selector '>'
  6. group Selector ','
