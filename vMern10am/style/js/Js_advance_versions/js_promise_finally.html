[+] Js Promise Finally
[ECMA Script 2018]

Promise Finally => It is Concept that runs over error and finally block runs in every status. 

Example :
            
            #Promise finally
            var p = new Promise((res,rej)=>{
                setTimeout(()=>{
                    //res("Promise is Resolved")
                    rej("Promise is Rejected")
                },1000)
            })
            p
            .then((data)=>{
                console.log(data)
            })
            .catch((error)=>{
                console.log(error)
            })
            .finally(()=>{
                console.log("Finally Called")
            })
