Adding padding in string 

            Example :

            String padding
            var a = "Ducat Educat Noida"
            var b = a.padStart(50,"-")
            var c = a.padEnd(50,"-")
            console.log(b)
            console.log(c)

            for(let i = 1;i<=10;i++){
