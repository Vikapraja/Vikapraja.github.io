[+] Js Nullish Coalescing (??)
[ECMA Script 2020]

Nullish Coalescing(??) : => this operator execute second statement only if first statement result is undefined or null else only first statement will execute 
            Syntax :
            
                statement1??statemnet2
            
            

            
            Example :

            Example [Using ??] :
            
                console.log(10||20)             //10
                console.log(0||20)              //20
                console.log(false||20)          //20
                console.log(""||20)             //20
                console.log(undefined||20)      //20
                console.log(null||20)           //20
                console.log(false||"N/A")       //N/A
            
                console.log(10??20)             //10
                console.log(0??20)              //0
                console.log(false??20)          //false
                console.log(""??20)             //""
                console.log(undefined??20)      //20
                console.log(null??20)           //20
                console.log(false??"N/A")       //false
