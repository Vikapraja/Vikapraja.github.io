#Examples :
  1. document.anchors: Returns all < a> elements that have a name attribute
  2. document.applets: Deprecated
  3. document.baseURI: Returns the absolute base URI of the document
  4. document.body: Returns the < body > element
  5. document.cookie: Returns the document's cookie
  6. document.doctype: Returns the document's doctype
  7. document.documentElement: Returns the < html> element
  8. document.documentMode: Returns the mode used by the browser
  9. document.documentURI: Returns the URI of the document
  10. document.domain: Returns the domain name of the document server
  11. document.domConfig: Obsolete
  12. document.embeds: Returns all < embed> elements
  13. document.forms: Returns all < form> elements
  14. document.head: Returns the < head> element
  15. document.images: Returns all < img > elements
  16. document.implementation: Returns the DOM implementation
  17. document.inputEncoding: Returns the document's encoding (character set)
  18. document.lastModified: Returns the date and time the document was updated
  19. document.links: Returns all < area> and < a> elements that have a href attribute
  20. document.readyState: Returns the (loading) status of the document
  21. document.referrer: Returns the URI of the referrer (the linking document)
  22. document.scripts: Returns all < scripts > element
  23. document.strictErrorChecking: Returns if error checking is enforced
  24. document.title: Returns the < title > element
  25. document.URL: Returns the complete URL of the document
