#JsArray Destructuring

  • =>Array destructuring in Js a way to extract values from arrays and assign them to variables.
  • =>it's useful when working with array that have a fixed number of elements and you only need to extract a subset of them.
  • #Note:
    1. Array destructuring works on iterable values, such as string and sets. It doesn't work on non-iterable values,such as null and undefined.
    2. If the number of variables specified on the left-hand side of the assignmenst is greater than the number of elements in the array on the right-hand side, only the fist N variable will be assigned values.
#Syntax :>->Use square brackets to store the variable name, and then assign it to the name of the array. 
            Like :
            
    
                
  • Array :  var array_name=[1,2,3,.....elements]
    • 
                
  • Array Destructuring :  [a,b,c,.....variables]= array_name
    • 
                
  • Elements in variable :  a=1, b=2, c=3 .....
    • 
            

                        
#Example :


                        var amp=[1001,"Vivek","student",35000,"Faridabad",Hariyana]
                        
var [id,name,desg,salary,city,state]=emp

            Now you can use Arrays values by variable :
                (InNode)    console.log(`
                                        Id          :   ${id}
                                        Name        :   ${name}
                                        Designation :   ${desg}
                                        Salary      :   ${salary} 
                                        City        :   ${city}
                                        Satate      :   ${state}              
                
                                        `)
