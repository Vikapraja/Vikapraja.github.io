#Object in JavaScript

  • =>Symbol: {...}is for object like symbol [...] is for array
  • =>Object store information in pair of keys and values.
  • =>Object are Mutable(Changeable).
  • =>Object can have duplicate values but keys must be unique.
  • =>If we use duplicate keys then only key will in work.
