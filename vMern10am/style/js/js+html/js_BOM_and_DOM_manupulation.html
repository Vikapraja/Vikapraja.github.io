#BOM & DOM Manupulation in JavaScript
  • BOM=Browser Object Model
  • DOM=Document Object Model

âž¡ BOM & DOM refers to structure related to JavaScript intract with the browser and web page content.

  1. BOM [Browser Object Model]
    • =>Bom allow JavaScript to intract with the browser itself.
    • =>It provide objects to intract with the browser window and control browser related properties.
    • Key features of BOM :
      1. Window Object
        • It is the main Object of the BOM which represent the browser window.
        • All other BOM object like Navigator, Screen, History, Loacation and Document etc. are properties of window object.
      2. Navigator
        • This Object provide information about the browser-name, virsion etc.
      3. Screen Object
        • This Object contains information about the user's screen such as : screen-width, screen-height etc
      4. Location
        • It object provide details about current URL and allow us to redirect browser to another page.
      5. History
        • This object present session-history of the browser tab.
        • It allow us to navigate backword and farword between pages.
  2. DOM [Document Object Model]
    • =>DOM represent structure of HTML and XML document.
    • =>It is an API for accessing and modifying the structure and contents of web page.
    • Key feature of DOM :
      1. Document Object
        • It is root of the DOM with representing entire HTML document.
      2. Node
        • DOM representing HTML Element as a tree of Nodes.
        • JavaScript can intract with these Node to manupulate the content and structure of the web-page.
      3. Element Access
        • We can select elements from DOM using methods like getElementById(), getElementByClassName(), getElementByTagName() and getByQuerrySelector() all.
      4. Event Handling
        • DOM allow for handling user's event like clicks, key_press, form_submition etc.through event listeners.
      5. Modifying Content
        • We can dynamically update and modify content files, attributes and structure of web pages
    #Key Differance between BOM & DOM :
  1. Scope : [BOM dills with the browser window and its properties while DOM is consult with the web page content.]
  2. Object : [BOM Object include window Navigator, Location, History object while DOM object include html elements and document stucture. ]
  3. Intraction : [BOM lets you control browser extends while DOM allows us to modify the web page.]
    ##DOM oprations : With help of DOM we can perform following operations :
  1. We can change the html doocument using DOM
  2. We can change html elements and attributes in the page.
  3. We can change all CSS-style in the page.
  4. We can create, update or delete html elements and attributes in the page.
  5. We can react over existing html events in page.
  6. we can create our own event and react on them in page.
  7. We can perform time-based event in the web page.
