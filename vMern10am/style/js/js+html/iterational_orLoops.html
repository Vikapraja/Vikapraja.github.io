#2)Loops/Iterational Statemants

=>Looping are used to repeatedly run a set of instruction(code /statements) while a given condition is true or false.

#Types =>

  1. for Loop
  2. for in Loop
  3. for of Loop
  4. while Loop
  5. do-while Loop
  6. Nested Loop
#Another Types Loop:
  • Entry Controled Loop [for and while loop]

    =>That loop tests the loop's condition at the loop's beginning before Execution of loop's body.

  • Exit Controled Loop [do-while loop]

    =>That loop execute the loop's body at least once before checking the loop's condition.
