#Map in JavaScript

  • =>Collection of item but in pair of keys and values.
  • =>Key must be unique but values may be duplicate in Map.
  • =>If we use duplicate keys in map then only last key will be used in Map.
  • => Map support indexing:
  • Function of Map :
    1. Set( array )
    2. Array.from( Set )
    3. add()
    4. delete()
    5. clear()
    6. size()
    7. has()
    8. vallues()
    9. values()
    10. keys()
    11. entires()
#Syntax :

                                    ])
