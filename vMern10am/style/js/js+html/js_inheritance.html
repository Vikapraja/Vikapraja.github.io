#Inheritance in JavaScript

  • =>OOPs Approach provide a feature to reuse all member of any previous existing class in a new class.
  • =>e.g If a class named B inherit class A then class B can reuse all member of class A. So class B is called Child-Class, derived-class or sub-class and class A is called Parent, Super or Base-class.
  • extends keyword are be used.
  • => Types Of Inheritance :
    1. Single Inheritance >

      When a single child class inherit a single parent class.

    2. Multi-Level Inheritance >

      when a child class inherit another child class

    3. Hierarchical Inheritance >

      When a single parent class inherit by more then one child class

    4. Multiple Inheritance >

      JavaScript Doesn't support Multiple Inheritance.

  • =>NOTE :
    1. Js doesn't support multiple-inheritance.
    2. a child-class can access only those members of parent-class which child-class doesn't have. 
#Syntax :
                        class parent{
                            ............
                        }

                        class child extends parent{
                            ............
                        }
