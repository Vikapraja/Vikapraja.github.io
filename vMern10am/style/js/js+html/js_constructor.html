#Constructor in JavaScript class

  • =>Constructor is special function that no need to call it. It called automatically when object is decleared with help of new keyword.
  • =>A special member method of class with provide memory initialization to a class object.
  • =>It can be parameterised.
  • =>If we not provide definition to a constructor in class then compiler(JIT) provide definition to constructor.
  • =>We cann't return any specific value from a constructor.
#Syntax should be like function:
                                        class class_name{
                                            
                                            constructor(parameters){
                                                ........
                                            }
                                            
                                            method1(){
                                                ........
                                            }
                                        }
