#Nested-Loop

=>Nested-loop means loops within loop.

#Syntax :
(A)Nested for 
            for(initialization;condition;updation){
                ...
                ...
                for(initialization;condition;updation){
                ....
                ...
                }
                ...
                while(condition){
                    ...
                    ...
                }
            }
(B)Nested while 
            while(condition){
                .....
                .....
                while(condition){
                    .....
                    ...
                }
                ....
                ....
                for(initialization;condition;updation){
                    ...
                    ...
                    for(initialization;condition;updation){
                    ....
                    ...
                    }
                    ...
                    while(condition){
                        ...
                        ...
                    }
                }
            }
