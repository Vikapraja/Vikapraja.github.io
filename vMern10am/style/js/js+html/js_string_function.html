#JS String Function / Methods

  • =>String Methods are those method that are use to do operation on String.
  • sub-string :some part of given string like "Vivek" is sub-string of "Vivekanand"
  • #Functions of string:
    1. CharAt() : [it takes index as argument return charactor at any specific position(index) / use indexing a[index]]
    2. CharCodeAt() :[return unicode of specific given index argument's item NOTE:it takes 1st char if no arguments.]
    3. indexOf() :[it takes sub-string as agrgument and returns 1st index of any sub-string in string, returns -1 if sub-string not found. ]
    4. lastIndexOf() :[it takes sub-string as argument and return last index of sub-string, returns -1 if sub-string not found in str.]
    5. search() :[it takes substring as argument and return 1st index of given sub-string in str and return -1 if sub-string not found]
    6. includes() :[it takes sub-str as argument and check wether a sub-string is exist in string or not. return true/false]
    7. concat() or '+' operator :[it takes multiple str as argumennt and put them sequencely(sideBySide). OR you can use + operator to concat strs]
    8. startsWith() :[Check wether a string start with specific sub-string or not. Rreturns true or false ]
    9. endsWith() :[check wether a string end with specific sub-string or not. Rreturns true or false]
    10. toUpperCase() :[It return Upper case{capital letters} of string's char]
    11. toLocaleUpperCase() :[It return Upper case{capital letters} of string's char]
    12. toLowerCase() :[It return Lower case{small letters} of string's char]
    13. toLocaleLowerCase()[It return Lower case{small letters} of string's char]
    14. substr() :[returns a sub-string form given position(index) to number of chars.it takes position and number(inclusive) of chars as argument . ]
    15. subString() :[returns a sub-string form given position(index) to number of chars.it takes position and number(exclusive) of chars as argument .]
    16. slice() :[It returns sub-string from given index as argument till end.]
    17. repeat() :[Create number of copies of a String.it takes number repeatation as argument.]
    18. split() :[Used to split() to convert a string into array of sub string.takes " " as argument]
    19. trim() :[Removes all leading white-space chars from string's both side.without argument]
    20. trimStart() :[Removes all leading white-space chars from string's left side.without argument]
    21. triEnd() :[Removes all leading white-space chars from string's right side.without argument]
    22. localCompare() :[used compare to string return 0 if both string are same
      return 1 if first string comes last in alphabetical order
      returns -1 if first strings comes first in alphabetical order]
#Syntax : 
            string.Function(argument)
