#for Loop

=>for used when execution of a statements(block of code) depend on a condtion.
when condition is true then Statemants are executed otherwise not.

# Syntax : 
                        for(intialization;condition;updation){
                            ......
                            ......statements
                            .....
                        }

#note : {} are optional if we have only one statement in statement-block.
