#JS Async/Await_and Exception Handling

Example :

            async function getPromise() {
                    try {
                        console.log(await p1)
                        console.log(await p2)
                        console.log(await p3)
                    } catch (error) {
                        console.log(error)
                    }
                }
                getPromise()

Some Methods :

  1. The global fetch() method starts the process of fetching a resource from the network, returning a promise that is fulfilled once the response is available.
  2. A fetch() promise only rejects when a network error is encountered (which is usually when there's a permissions issue or similar). A fetch() promise does not reject on HTTP errors (404, etc.).
  3. Instead, a then() handler must check the Response.ok and/or Response.status properties.
