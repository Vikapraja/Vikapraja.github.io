#JS Function Curring
#Function Curring : =>it is a technique of evaluating function with mutliple argumnets into sequance of function with single argument
=>i.e when a function,instead of taking all argumemnts at one time,it takes the first argument and return a new function which takes the second argument and return a new function which takes the third one and so on 

 Example :
                function sum(num1){
                        return function(num2){
                            return function(num3){
                                return function(num4){
                                    return function(num5){
                                        return num1+num2+num3+num4+num5
                                    }
                                }
                            }
                        }
                    }
                    
                    let sum = (num1)=>(num2)=>(num3)=>(num4)=>(num5)=>num1+num2+num3+num4+num5
                    console.log(sum(100)(200)(300)(400)(500))
