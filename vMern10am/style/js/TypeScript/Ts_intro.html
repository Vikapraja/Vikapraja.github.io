Key-features of TypeScript :

  1. Static/Strict typing
  2. Type Inference
  3. Interface
  4. ENUMS
  5. Advance IDE Support
  6. Access Modify
  7. Compatibility of JavaScript
  8. Names Space and Module
  9. Asynchronous Programming
  10. Germics
  • Static/Strict typing : TypeScript is statically typed which means we can defines type of variable, function, parameter and reurn values etc.
    => This insure type safety and reduce run time error.
  • Type Inference : TypeScript can automatically infere type even we don't explicitly explain.
  • Interface : Interface allow us to define structure for objects making it easier to inforce specific shape and properties for objects.
  • ENUMS : JavaScript provide an Enumus type which allow us to define a set of named constant
    =>This is specialy used for define values that will be use multiple time in a constant.
  • Advance IDE Support : TypeScript provide enhenced auto complition, navigation and refactoring capabilities. Due to it's strong typing and tooling support.
  • Access Modify : TypeScript support public, private and protected access modifier for class members making easier to control how data is accessed and modified.
  • Compatibility of JavaScript : TypeScript is fully Compatible of JavaScript libraries and frameworks.
  • Names Space and Module : TypeScript offers name space and libraries as well as ES6 module support making code moduler and manageable.
  • Asynchronous Programming : TypeScript support 'async' and 'await' keyword making it easier to work with Asynchronous code in a relatable way.
  • Germics : TypeScript supports germics to allow to create reuseable components. That works with rather than single work.
