TS OOPs and Abstraction & Encapsulation

#Ts OOPs and Abstraction & Encapsulation:

  1. OOPs(Object oriented Programming) .
  2. Objct and class.
  3. Abstract Classes =>
    1. Abstract classes are a fundamental concept in TypeScript, they enable us to define classes that represent a base from which other classes may be derived.
      => They serve as a blueprint for other classes without being instantiated themselves.
  4. Access Modifiers : Means object's access type.
    1. public =>object accessable everywhere
    2. private =>Objects limited accessable
    3. protected =>object Not accessable

            
Example :

Class and Object: 

            class Test{
                    private a:number
                    private b:number
                    private sum:number

                    setData(a:number,b:number){
                            this.a = a
                            this.b = b
                            this.sum = a+b
                    }

                    display(){
                            console.log(`${this.a} + ${this.b} = ${this.sum}`)
                    }
            }

            var obj = new Test()
            obj.setData(10,20)
            obj.display()

            console.log(obj.a)
            console.log(obj.b)
            console.log(obj.sum)



Abstract Class :

                abstract class TaxCalculation{
                        protected name:string
                        protected price:number
                        protected taxType:string
                        protected tax:number
                        protected total:number
                        display(){
                                console.log(`
                                                Product Name   :       ${this.name}
                                                Tax Type       :       ${this.taxType}
                                                Price          :       ${this.price}
                                                Tax            :       ${this.tax}
                                                Total          :       ${this.total}
                                        `)
                        }
                    
                }
                class Product1 extends TaxCalculation{
                        calulcate(name:string,price:number){
                                this.name = name
                                this.taxType = "VAT"
                                this.price = price
                                this.tax = this.price*30/100
                                this.total = this.price+this.tax
                        }
                }
                
                class Product2 extends TaxCalculation{
                        calulcate(name:string,price:number){
                                this.name = name
                                this.taxType = "GST"
                                this.price = price
                                this.tax = this.price*18/100
                                this.total = this.price+this.tax
                        }
                }
                var petrol = new Product1()
                var laptop = new Product2()

                petrol.calulcate("Petrol",50000)
                laptop.calulcate("Laptop",50000)

                petrol.display()
                laptop.display()
