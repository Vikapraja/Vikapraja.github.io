TS Interface vs Obj-Type alias

#Ts Interface vs Obj-Type alias :

  1. In case of type alias we have to define alias single time, but in case of interface we can define it mulitple times with same name
  2. We can use a obj-type alias with another obj-type alias.
  3. We can inherti a interface to another interface like inheritace.

            
Example :

Obj-Type Alias :
                type test1 = {
                        a: number,
                        b: number
                }
            
                var obj1: test1 = {
                        a: 10,
                        b: 20
                }
                console.log(obj1)


Interfaces :
                interface test2 {
                        a: number,
                        b: number
                }
                interface test2 {
                        c: number,
                        d: number
                }
                var obj: test2 = {
                    a: 10,
                    b: 20,
                    c: 30,
                    d: 40
                }
                console.log(obj)
                


Union of Obj-alias Type :
                type A = {
                        a: number,
                        b: number
                }
                type B = A & {
                    c: number,
                    d: number
                }
                var obj:B = {
                    a:10,
                    b:20,
                    c:30,
                    d:40,
                }
                


Inherit Interfaces to other interface by 'extends' keyword like inheritace :
            
                    interface A{
                            a: number,
                            b: number
                    }
                    interface B extends A {
                            c: number,
                            d: number
                    }
                    var obj:B = {
                            a:10,
                            b:20,
                            c:30,
                            d:40,
                    }

                    console.log(obj)
