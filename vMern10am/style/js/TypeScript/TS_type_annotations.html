Type Annotation or Strict Typing

#Type Annotation or Strict Typing :

=> Means after defining datatype of a variable, you can'nt store other datatype's value.
=> Note : a variable can store different datatype by any-datatype. 
            

                
#Defining datatype of variable :

                    =>Syntax :  var var_name:datatype(name) = value
                


            Example :

                var a:number = 10
                console.log(a)
                // a = "Ducat"  //error
                console.log(a)
                // a=true          //error
                console.log(a)
                // a=[10,20,30,40,50,60]   //error
                console.log(a)
