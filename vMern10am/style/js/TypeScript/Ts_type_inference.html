TS Type Inference

#Type Inference :

=> Means after assigninig value of a varibale , variable decides datatype automatiaclly by value type.
=> And after it Variale can'nt store Other datatype's value.
=> Note : a variable can store different datatype by any-datatype. 
            

            Example :
                            var a = 10
                            console.log(a)
                            //a = "Ducat" //error
                            console.log(a)
                            //a=true //error
                            console.log(a)
                            //a=[10,20,30,40,50,60]   //error
                            console.log(a)
