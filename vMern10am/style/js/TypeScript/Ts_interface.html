TS Imterface

#Ts Imterface :

  1. use of 'interface' keyword.
  2. An interface is a syntactical contract that an entity(object) should conform to.
  3. In other words, an interface defines the syntax that any entity must adhere to.
  4. Interfaces define properties, methods, and events, which are the members of the interface. Interfaces contain only the declaration of the members. It is the responsibility of the deriving object to define the members. It often helps in providing a standard structure that the deriving object would follow. 

            
Example :


                interface EmployeeType{
                            id:number,
                            name:string,
                            dsg:string,
                            salary:number,
                            city:string,
                            state:string
                    }
    
                    var emp: EmployeeType = {
                            id: 1001,
                            name: "Nitin Chauhan",
                            dsg: "Trainer",
                            salary: "135000",
                            city: "Faridabad",
                            state: "Haryana"
                    }
                    console.log(emp)
