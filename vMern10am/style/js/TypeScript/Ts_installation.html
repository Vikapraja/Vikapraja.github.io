TypeScript Installation

#Requirement to install TS:

  1. NodeJs sould be installed.
  2. Node provide 'npm' also.It means node package manager.
    => it is a CLI tool. which helps us to install,unstall and update JavaScript's external module and packages.

Installation process :

  1. To checking npm is avaialable or not. On Cmd ' npm -v '
  2. Go to Command Prompt (Cmd) and run command 'npm i -g typescript' to install TS.
    • npm => node package manager
    • -g => making installation globaly
    • i => Install
#Execution of TS code :
  1. Create TypeScript file with .ts extention and Write Ts Code (mostly common in JS Code).
  2. To run Ts file go to cmd at file path and type tsc file_name.ts hit enter. it command also convert ts file to js file.
    =>Or node file_name.ts hit enter.
Note:Ts can not run with HTML.

#command to change ts to js :

  • Create js file, name with same of ts file at same path.
  • And compile ts file on cmd ts file_name.ts + enter
  • You see you ts code also in js file.
  • Watch mode done (ts to js ) automatically as you write ts code on ts file.

# Watch mode :

  • Run ts file on cmd with -w like tsc file_name.ts -w
  • Error : If you run wtch mode first time you could find error like runnig script is disable on this system.
  • Solution : Go to Power shell and run it as admin and run follwing command :
    1. To allow runnig script in Windows ExecutionPolicy RemoteSigned hit enter and then y hit enter.
    2. To block runnig script in Windows on shell cmd Set -ExecutionPolicy Restricted + enter
