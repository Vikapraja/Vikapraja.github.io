#CSS Text

  • => CSS text properties are used to style texts.
  • => CSS text properties type with it's pocible values:
    text-properties: value ;
    • font-size
      1. (int)px
      2. (float)rem
      3. (int)vw
      4. large
      5. medium
      6. etc.
    • text-align
      1. center
      2. left
      3. right
      4. justify
      5. end
      6. start
      7. inherit
      8. etc.
    • text-transform
      1. uppercase
      2. lowercase
      3. capitalize
    • text-decoration
      1. underline (int)px
      2. overline (int)px
      3. line-through (int)px
    • font-style
      1. italic
      2. intialize
      3. inherit
      4. etc.
    • font-weight
      1. bold
      2. bolder
      3. lighter
      4. etc
    • font-family
      1. cursive
      2. sans-serif
      3. arial
      4. etc.
    • text-shadow
      1. +-(int)px +-(int)px (int)px color
      2. [NOTE: +ve value for right side shadow & -ve value for left side shadow]
