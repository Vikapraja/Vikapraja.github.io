#CSS Background

  • =>CSS Background properties takes colors as its value.
  • =>Background properties types and its values :
    1. background
      • colors
      • linear-gradiant
      • radial-gradiant
      • etc
        •
    2. background-color
      • colors
        •
    3. background-image
      • url(...)
        •
    4. background-repeat
      • repeat
      • no-repeat
      • repeat-x
      • repeat-y
        •
    5. background-position
      • fixed
      • left
      • right
      • bottom
      • top
        •
    6. background-size
      • auto
      • cover
      • contain
      • (int)px
      • etc
        •
    7. background-blend-mode
      • multiply
      • overlay
      • etc
        •  
#Syntax :
                
    
                    
  1. 
                    background: [color/img as value];
                    #OR
                    background-color: [colors/img as value];
                    
    2. 
                    
  2. background: linear/radial-graduant(color1 color2 color3 ....);
    3. 
                    
  3. background-image: url(...);
    4. 
                    
  4. background-repeat: value;
    5.
