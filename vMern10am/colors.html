#Colors in CSS

  • =>Every Color is combination of RGB(RED,GREEN,BLUE)
  • Memory Space according color
    Color Memory Space(bit)
    red 0-255
    green 0-255
    blue 0-255
    NOTE:In color's values hexadecimal numbers are taken because it holds more value in short space or short number.
  • #Css color properties values :
      => color : (value);
    1. color-name Examples=> red, green, blue, navy, ligthgreen etc.
    2. hexa-munber Examples=> #2a3baa, #ffff, #2a2a2a etc
    3. rgb(red, green, blue) Examples=>rgb(150, 143, 60), rgb(90, 55, 89) etc
    4. hsl(hue, seturation, lightness) Examples=>hsl(1, 50%, 50%), hsl(30, 50, 25%), etc.
    5. hsla(hue, seturation, lightness, alpha) Examples=>hsla(1, 50%, 50%, 1), hsla(40, 50%, 60%, 1), etc
#Syntax : 
            color: [color-value];
