React Installation & Project Setup

Requirement To install React :

  1. Installed nodeJS
  2. Installed npm (Node Packege Manager)
  3. TypeScript

React Installation :

  1. Go to path where you want start project and open CMD with this path.
  2. In CMD Run this command > npm i -g create-react-app + enter
    • npm : node package manager
    • i : Install
    • -g : For Globly installation
    • create-react-app : React module/ Package name

    Note :it is react server installation
  3. After React installation In CMD(To Create app of Project) > npx create-react-app 'project/app-name' + enter
    • npx : node package execution
    • crete-react-app : To create project's app
    • 'app-name' :React Project/app name
      Note:App-name should be without space+hyphon and only allow small letters.
  4. Happy hacking! : sign of right installation
  5. After Creating app Go to app Folder with cmd (To Start Project) >npm start + Enter
    NOTE-U can see react Symbol

After Creating Projecct/app Auto Created Folders & files :

  1. node_modules (folder)
  2. public (folder)
  3. src (folder)
  4. package (json-file)
  5. package-lock (json-file)
  6. .gitignore (txt-file)
  7. README (markdown_file.md)
  1. node_modules (folder) :
    • It is contain all nessesery data for react project.
    • All dependencies are on it , it should never be delete nor project will not work.
    • It has bable compiler that convert react code to plane JavaScript code.
      => (browser not understand react code)
  2. public (folder) : it is folder where we code .5%
  3. src (folder) : It is folder where we doing coding 99%
  4. package (json_file) :
    • It contains all information about project & module in json format like :project-name, version, dependencies etc
    • and also contains project related commands like start, build, test etc .
  5. package-lock (json_file) : It is React dependencies file .
    Note :React is dependencies of project and package-lock is dependencies of React.
  6. .gitignore (txt_file) : It suggest that which file should be upload on git and which should not.
    =>Without it git never work on this project.
  7. README (markdown_file.md) : it is for git hub.

Creating Project :

  1. After installing react and creating app make empty to public and src folder. (delete everything from public and src)
  2. This is nessesery to create firstly index.html in public folder and index.js in src folder.(it are entry/lander pages)
