Rendering Items In React

Why need reandering (react) ?

  • To sending object from app>src>index.js to app>public>index.html
  • React rendering function from src>index.js return/sends single item(element) only to public>index.html.
  • In term of return/sending two or more item(element) from render, React gives Errors.
  • We can do multiple item/object/element in a single item/element/object then render.
  • So...this is need of react render

Render Types : [ In app>src>index.js ]

  1. By Use of Array [.,.,.]
  2. By Use of div
  3. By use of 'React.Fragment' tag < React.Fragment>....< /React.Fragment>
  4. By empty anguler bracket < >....< /> [React Fragment syntactic Suger Format]
  1. By Use of Array : By Array we can make single item/object/element of multiple Object/item/elemets.
    =>Then render will not give error. [ But it is booring]
    Like : 
            root.render(
                    [
                        < h1>Hello World!!!< /h1> ,
                        < h2>This is first react program< /h2>
                    ]
                )
  2. By Use of div : By div we can make single item/object/element of multiple Object/item/elemets.
    =>Then render will not give error.
    => But div may make disturb project.
    Like : 
            root.render(
                    < div>

                        < h1>Hello World!!!< /h1>

                        < h2>This is first react program< /h2>

                    < /div>
                )
  3. By Use of < React.Fragment>....< /React.Fragment> : By 'React.Fragment' tag we can make single item/object/element of multiple Object/item/elemets.
    =>Then render also will not give error . [ it not make any disturbens]
    Like : 
            root.render(
                    < React.Fragment>

                        < h1>Hello World!!!< /h1>

                        < h2>This is first react program< /h2>
                    
                    < /React.Fragment>
                )
  4. By Use of React-Fragment-Syntactic-suger-Format (Empty anguler bracket) < >....< /> : By 'React.Fragment' tag we can make single item/object/element of multiple Object/item/elemets.
    =>Then render also will not give error . (It is best way to rensering multiple items)
    Like : 
            root.render(
                    < >

                        < h1>Hello World!!!< /h1>

                        < h2>This is first react program< /h2>
                    
                    < />
                )
