Rendering Items In React

What is component ?
=> Parts of a project

  1. Components Allow developers to beak-down complex user interface (UI) into smaller, reuseable and Isolated piece of UI.
  2. Each Components represents part of UI and can include its own logic , structure and styling.
  3. This Moduler approach simplifies the development process by making code more readable , maintainable and easier to debug.

Rule for component :

  1. Component file should be in 'App>src'
  2. Component's name should be start with capital letter.
  3. Component is also js or jsx file.
  4. Component shoul be export at created time and should be import at use time.

Types of Components :

  1. Class Components
  2. Functional Components
  1. Class Components :
    • After creating component file (js/jsx) 'react' should be import.
    • Class component are based on class as OOPs.
    • class should be inherit to 'component'.
    • In class render() is nessesery and in render() , return() is nessesery.
    • In return() object/item/element should be defined.
    • And class should be export default at created time. To import from where we want to use this class.
    Example : 
        App/src/ClassComponent/Sample.jsx

    import React, { component } from "react"

    class Sample extends component{
                    render(
                     return(
                                <>
                                    < h1> Class Component Example < /h1>
                                    < h2> This is Sample class component < /h2>
                                < />
                            )
                        )
                }
        
    export default Sample
    
    
    Or :

    import React, { component } from "react"

    export default class Sample extends component{
                    render(
                     return(
                                <>
                                    < h1> Class Component Example < /h1>
                                    < h2> This is Sample class component < /h2>
                                < />
                            )
                        )
                }
    Importing class component : (In other js/jsx file/page) 
        import Sample form "./ClassComponent/Sample"

    < Sample/>       // where you want use component
  2. Functional Components :
    • After creating Component (js/jsx file) 'react'should be import.
    • Functinal Component are based on function.
    • In Function render() is nessesery and in render() , return() is nessesery.
    • Function should be export default at created time. To import where we want to use Functional Component.
    Example :
    1. By use of reguler function 
               (App/src/FunctionalComponent/SampleF) 
                        
        function SampleF(){
            return(
                    <>
                        < h1> Functinal component Example < /h1>
                        < h2> This is SampleF Functinal Component < /h2>
                    < />
                )
            }
        
        export default SampleF
                        
        Or
                        
        export default function SampleF(){
            return(
                    <>
                        < h1> Functinal component Example < /h1>
                        < h2> This is SampleF Functinal Component < /h2>
                    < />
                )
            }
      Importing Functional component : (In other js/jsx file/page) 
          import Sample form "./FunctionalComponent/Sample"

    < SampleF/>       // where you want use component
    2. By use of Anonymous Function : 
           (App/src/FunctionalComponent/SampleF) 
                        
    const SampleF = function (){
        return(
                <>
                    < h1> Functinal component Example < /h1>
                    < h2> This is SampleF Functinal Component < /h2>
                < />
            )
        }
    
    export default SampleF
    3. By use of Fat-arrow function : 
           (App/src/FunctionalComponent/SampleF) 
                        
    const SampleF = ()=>{
