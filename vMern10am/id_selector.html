#Id selector

  • =>The id selector uses the id attriute of an HTML element to select a specific element.
  • =>The id of an element is unique within a page,so the id Selector is used to select one unique element !.
  • =>To select an element with a specific id, write a hash(#) charactor. followed by the id of the element.
