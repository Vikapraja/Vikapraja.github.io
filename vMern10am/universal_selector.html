# Universal Selector( * ) in CSS =>

  • The Universal SElector indicated by an astrisk(*) .
  • It selects everything in the document.
  • if * is chained using a descendant combinator,it select everything inside that ancestor element.
  • For Example, *p select all the nested element(elements under p) in the 'p' tag/element.
    • #Syntax: 
                    *{
                    .......
                    style-code(css-code) ;
                    .......
                 }
